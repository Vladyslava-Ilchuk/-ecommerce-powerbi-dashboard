# 📊 E-Commerce Power BI Dashboard (Olist Dataset)

## 📌 Project Overview

This project presents the development of an interactive **Power BI dashboard suite** for a Brazilian e-commerce company based on the **Olist dataset**.

The objective of the project is to analyze business performance across multiple operational domains and transform raw transactional data into actionable business insights.

The solution includes **four interconnected dashboards** focusing on Sales, Logistics, Payments, and Customer Feedback, designed with a custom UX/UI structure to support intuitive navigation and executive-level decision-making.

---

## 🎯 Business Objective

The main goal of this project is to provide a comprehensive analytical system that enables stakeholders to:

- Monitor sales performance and revenue trends  
- Evaluate logistics efficiency and delivery performance  
- Analyze payment behavior and transaction success rates  
- Understand customer satisfaction and feedback patterns  

The dashboards are designed to support data-driven decision-making across multiple business departments.

---

## 🏗️ Data Source

The project is based on the **Olist Brazilian E-Commerce dataset**, which includes:

- Orders data  
- Customers data  
- Payments data  
- Product data  
- Logistics and delivery information  
- Customer reviews and feedback  

---

## 📊 Dashboard Modules

### 1. 📈 Sales Dashboard
- Revenue trends over time  
- Product category performance  
- Regional sales distribution  
- Year-over-year comparisons (DAX-based KPIs)  

### 2. 🚚 Logistics Dashboard
- Delivery time analysis  
- Late delivery rate  
- Shipping performance by region  
- Order fulfillment efficiency  

### 3. 💳 Payments Dashboard
- Payment method distribution  
- Transaction success vs failure rates  
- Average order value (AOV)  
- Monthly payment trends  

### 4. ⭐ Feedback Dashboard
- Customer satisfaction analysis  
- Review score distribution  
- Correlation between delivery time and ratings  
- Negative feedback drivers  

---

## ⚙️ Technical Implementation

This project was developed using **Power BI Desktop** with a strong focus on data modeling and analytical design.

Key technical components include:

- **Power Query** for data cleaning and transformation  
- **DAX (Data Analysis Expressions)** for KPI calculations and time intelligence  
- Relational data modeling across multiple tables  
- Custom measures for year-over-year comparisons  
- Dynamic filtering and slicers for interactive exploration  
- Navigation system for seamless dashboard switching  

---

## 🎨 UX/UI Design Approach

A custom dashboard layout was designed to ensure:

- Clear separation between business domains  
- Intuitive navigation structure  
- Minimalistic and executive-friendly interface  
- Consistent visual hierarchy across all dashboards  
- Interactive filtering for deep analytical exploration

## 📈 Key Insights

The dashboard enables identification of:

- Seasonal sales patterns and revenue fluctuations  
- Logistics inefficiencies impacting customer satisfaction  
- Payment method preferences across regions  
- Strong correlation between delivery delays and negative reviews  
- Regional disparities in e-commerce performance
